# Промпт для DeepSeek (краткая версия для копирования)

---

Создай профессиональную презентацию для защиты дипломного проекта на тему "Мобильное приложение для отслеживания женского здоровья WomenHealthTracker".

## О проекте:

**Тип:** Android-приложение на Kotlin
**Технологии:** Kotlin, Android SDK, Firebase (Authentication + Firestore), Material Design
**Целевая платформа:** Android (minSdk 23, targetSdk 34)

## Основные функции:

1. **Отслеживание менструального цикла:**
   - Календарь с цветовой индикацией фаз
   - Прогнозирование овуляции и ПМС
   - Трекинг симптомов (физических и эмоциональных)
   - Анализ задержки с рекомендациями
   - Статистика циклов

2. **Режим беременности:**
   - Счетчик шевелений плода
   - Трекинг симптомов беременности
   - Запись визитов к врачу
   - Хранение фото УЗИ
   - Информация о размере малыша по неделям
   - Статистика беременности

3. **Режим менопаузы:**
   - Отслеживание симптомов менопаузы
   - Виджет поддержки
   - Статистика симптомов

4. **Интеллектуальные прогнозы:**
   - Алгоритм CalendarPredictor для прогнозирования начала цикла, овуляции, ПМС, энергии и симптомов
   - Уровень уверенности 45-90% в зависимости от регулярности

5. **Умные уведомления:**
   - SmartNotificationScheduler для адаптивных уведомлений
   - Напоминания о важных событиях цикла

6. **Безопасность:**
   - Firebase Authentication
   - Cloud Firestore для синхронизации
   - Резервное копирование
