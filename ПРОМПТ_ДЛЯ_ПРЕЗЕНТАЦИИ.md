# Промпт для DeepSeek: Создание презентации для защиты проекта

## Промпт:

Создай профессиональную презентацию для защиты дипломного/курсового проекта на тему "Мобильное приложение для отслеживания женского здоровья WomenHealthTracker".

### Контекст проекта:

**Название проекта:** WomenHealthTracker - Мобильное приложение для отслеживания женского здоровья

**Тип проекта:** Android-приложение (нативное)

**Язык программирования:** Kotlin

**Целевая платформа:** Android (minSdk 23, targetSdk 34)

**Технологический стек:**
- Kotlin
- Android SDK
- Firebase Authentication (аутентификация пользователей)
- Cloud Firestore (облачное хранилище данных)
- Material Design Components
- AndroidX Libraries
- Gson (сериализация данных)
- Android Notifications API

### Основные функции приложения:

#### 1. Отслеживание менструального цикла (CYCLE_TRACKING)
- Календарь с цветовой индикацией фаз цикла
- Отметка начала менструации
- Прогнозирование овуляции (на основе 14 дня цикла с учетом индивидуальной длины)
- Прогнозирование ПМС (за 3-7 дней до начала цикла)
- Отслеживание симптомов (физических и эмоциональных)
- Анализ задержки менструации с рекомендациями
- Статистика циклов (средняя длина, регулярность, графики)

#### 2. Режим беременности (PREGNANCY)
- Настройка срока беременности (по дате последних месячных, дате зачатия или неделе)
- Счетчик шевелений плода
- Трекинг симптомов беременности
- Запись визитов к врачу с заметками
- Хранение фотографий УЗИ
- Виджет с информацией о размере малыша по неделям (сравнение с фруктами)
- Статистика беременности (график набора веса, симптомы по неделям)

#### 3. Режим менопаузы (MENOPAUSE)
- Отслеживание симптомов менопаузы (приливы, ночная потливость, изменения настроения и др.)
- Виджет поддержки с мотивирующими сообщениями
- Запись визитов к врачу
- Статистика симптомов (частота, интенсивность, графики)

#### 4. Трекер симптомов
- Физические симптомы (боли, выделения, вздутие, усталость, тошнота)
- Эмоциональные симптомы (настроение, энергия, стресс, концентрация)
- Активность (физическая активность, сон, упражнения)
- Привычки (вода, питание, лекарства, витамины)
- Быстрое добавление данных за день

#### 5. Интеллектуальные прогнозы
- Алгоритм CalendarPredictor для прогнозирования:
  - Начала цикла (на основе истории, за 2-3 дня)
  - Овуляции (14 день цикла ±2 дня)
  - ПМС (за 3-7 дней до начала)
  - Уровня энергии (0-100%, на основе исторических данных)
  - Симптомов (на основе истории)
- Уровень уверенности в прогнозах (45-90% в зависимости от регулярности циклов)

#### 6. Умные уведомления
- SmartNotificationScheduler для адаптивных уведомлений
- Уведомления о начале цикла (настраиваемое время и заблаговременность)
- Уведомления об овуляции
- Уведомления о ПМС
- Ежедневные напоминания о вводе данных
- Напоминания о воде и лекарствах

#### 7. Статистика и аналитика
- Статистика циклов (средняя длина, регулярность, графики)
- Симптомы по фазам цикла
- Энергия по фазам цикла
- Статистика беременности
- Статистика менопаузы
- Экспорт данных для врача

#### 8. Безопасность и синхронизация
- Firebase Authentication для безопасной аутентификации
- Cloud Firestore для облачного хранения данных
- Автоматическая синхронизация между устройствами
- Резервное копирование данных
- Конфиденциальность и безопасность данных

### Архитектура приложения:

**Структура проекта:**
- 50+ Activity классов для различных экранов
- Вспомогательные классы (Helpers):
  - CalendarPredictor (прогнозирование)
  - SmartNotificationScheduler (умные уведомления)
  - FirebaseAuthHelper (аутентификация)
  - FirestoreHelper (работа с базой данных)
  - BabySizeHelper (информация о размере малыша)
  - PregnancyWeekGuide (гид по неделям беременности)
  - SupportMessageHelper (сообщения поддержки для менопаузы)
  - CycleReportGenerator (генерация отчетов)

**Модели данных:**
- DayData (данные за день)
- PregnancyData (данные беременности)
- MenopauseData (данные менопаузы)
- FertilityData (данные фертильности)
- UserPreferences (настройки пользователя)
- UserGoal (цели пользователя)

### Технические особенности:

1. **Онбординг:** Процесс настройки при первом запуске (выбор целей, настройка цикла, выбор режима)

2. **Адаптивный интерфейс:** Material Design с поддержкой различных размеров экранов

3. **Офлайн-режим:** Возможность ввода данных без интернета с последующей синхронизацией

4. **Уведомления:** Использование AlarmManager и NotificationManager для точных уведомлений

5. **FileProvider:** Безопасный доступ к файлам (фото УЗИ)

6. **Разрешения:** Запрос разрешений для уведомлений, точных будильников
